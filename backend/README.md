# Setup

- Install Node.jS
- Pull this repo
- Open the project on vscode
- In your vscode terminal cd into the backend folder, that is, cd backend
- Run npm install
- Run npm run dev

  
# Base URL

- localhost: http://localhost:5000

- live link: https://snipy.onrender.com/
